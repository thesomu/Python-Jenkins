54class FIM(Barcode): 

55 """ 

56 FIM (Facing ID Marks) encode only one letter. 

57 There are currently four defined: 

58 

59 A Courtesy reply mail with pre-printed POSTNET 

60 B Business reply mail without pre-printed POSTNET 

61 C Business reply mail with pre-printed POSTNET 

62 D OCR Readable mail without pre-printed POSTNET 

63 

64 Options that may be passed to constructor: 

65 

66 value (single character string from the set A - D. required.): 

67 The value to encode. 

68 

69 quiet (bool, default 0): 

70 Whether to include quiet zones in the symbol. 

71 

72 The following may also be passed, but doing so will generate nonstandard 

73 symbols which should not be used. This is mainly documented here to 

74 show the defaults: 

75 

76 barHeight (float, default 5/8 inch): 

77 Height of the code. This might legitimately be overriden to make 

78 a taller symbol that will 'bleed' off the edge of the paper, 

79 leaving 5/8 inch remaining. 

80 

81 lquiet (float, default 1/4 inch): 

82 Quiet zone size to left of code, if quiet is true. 

83 Default is the greater of .25 inch, or .15 times the symbol's 

84 length. 

85 

86 rquiet (float, default 15/32 inch): 

87 Quiet zone size to right left of code, if quiet is true. 

88 

89 Sources of information on FIM: 

90 

91 USPS Publication 25, A Guide to Business Mail Preparation 

92 http://new.usps.com/cpim/ftp/pubs/pub25.pdf 

93 """ 

94 barWidth = inch * (1.0/32.0) 

95 spaceWidth = inch * (1.0/16.0) 

96 barHeight = inch * (5.0/8.0) 

97 rquiet = inch * (0.25) 

98 lquiet = inch * (15.0/32.0) 

99 quiet = 0

148class POSTNET(Barcode): 

149 """ 

150 POSTNET is used in the US to encode "zip codes" (postal codes) on 

151 mail. It can encode 5, 9, or 11 digit codes. I've read that it's 

152 pointless to do 5 digits, since USPS will just have to re-print 

153 them with 9 or 11 digits. 

154 

155 Sources of information on POSTNET: 

156 

157 USPS Publication 25, A Guide to Business Mail Preparation 

158 http://new.usps.com/cpim/ftp/pubs/pub25.pdf 

159 """ 

160 quiet = 0 

161 shortHeight = inch * 0.050 

162 barHeight = inch * 0.125 

163 barWidth = inch * 0.018 

164 spaceWidth = inch * 0.0275 

165 def __init__(self, value='', **args): 

166 value = str(value) if isinstance(value,int) else asNative(value) 

167 for k, v in args.items(): 

168 setattr(self, k, v) 

169 

170 Barcode.__init__(self, value) 

171 

172 def validate(self): 

173 self.validated = '' 

174 self.valid = 1 

175 count = 0 

176 for c in self.value: 

177 if c in (string_whitespace + '-'): 

178 pass 

179 elif c in string_digits: 

180 count = count + 1 

181 if count == 6: 

182 self.validated = self.validated + '-' 

183 self.validated = self.validated + c 

184 else: 

185 self.valid = 0 

186 

187 if len(self.validated) not in [5, 10, 12]: 

188 self.valid = 0 

189 

190 return self.validated 

191 

192 def encode(self): 

193 self.encoded = "S" 

194 check = 0 

195 for c in self.validated: 

196 if c in string_digits: 

197 self.encoded = self.encoded + c 

198 check = check + int(c) 

199 elif c == '-': 

200 pass 

201 else: 

202 raise ValueError("Invalid character in input") 

203 check = (10 - check) % 10 

204 self.encoded = self.encoded + repr(check) + 'S' 

205 return self.encoded

208 self.decomposed = ''

211 return self.decomposed

215 self._height = self.barHeight
