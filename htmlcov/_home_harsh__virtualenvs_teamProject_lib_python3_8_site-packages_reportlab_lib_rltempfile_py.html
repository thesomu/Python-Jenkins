7When our test suite runs as source, a script "test_foo.py" will typically 

8create "test_foo.pdf" alongside it. But if you are testing a package of 

9compiled code inside a zip archive, this won't work. This determines

12 

13'''

23def get_rl_tempdir(*subdirs):
